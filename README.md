# Smart Resume Parser

Smart Resume Parser is an NLP-based project that extracts structured information from unstructured CVs (resumes) and matches them with job requirements.

The project uses Transformer-based models and rule-based NLP techniques, and provides an interactive web interface built with **Gradio**.
